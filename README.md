# Temperature Converter (C Implementation)

A minimalist and efficient Command Line Interface (CLI) tool to perform temperature conversions between **Celsius**, **Fahrenheit**, and **Kelvin**.

## 🚀 Key Features
- **Accurate Conversions:** High-precision `float` math.
- **Robust Menu:** Built-in input validation.
- **Clean Architecture:** Modular design with specific functions for each formula.

## 🛠️ Installation & Usage
1. **Compile:** `gcc temp_converter.c -o temp_converter`
2. **Run:** `./temp_converter`

## 📐 Formulas Used
- **Fahrenheit:** `F = (C * 1.8) + 32`
- **Celsius:** `C = (F - 32) / 1.8`
- **Kelvin:** `K = C + 273.15`
